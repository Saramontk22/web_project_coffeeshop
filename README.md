# Café Tripletten

Este projeto de página web foi desenvolvido para apresenta um espaço criado para estudos e convivência,umm café especial.
Esta página inclui uma sesão com receitas, reserva de mesas e link para as redes sociais em seu rodapé.
O foco do projeto foi desenvolver habilidades na organização de arquivos CSS e utilização da metodologia BEM ( Block, Element, Modifier).

As Técnologias utilizadas foram: HTML5, CSS,Metodologia BEM, Organização de CSS com @import, e Normalize css.
Suas funcionalidades são a estruturação semântica com HTML, Layout responsivo, formulário de reservas, videos, links para as redes sociais e organização de css.

As melhoria a serem realizadas são a implementação de animações e transições para melhorar a experiência do usuário e tornar a página responsiva para dispositivos móveis.
